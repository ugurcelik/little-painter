Restore Purchases

If you bought premium content before, open the app settings and tap Restore Purchases. Make sure you are signed in with the same Apple ID used for the original purchase.

Premium Access

Single category purchases unlock that category permanently, including new drawings later added to the same category. All Access unlocks all current and future premium categories.

Photo Library Permission

Little Painter asks for photo library permission only when saving artwork to the device gallery. If saving does not work, check iOS Settings and make sure photo access is allowed for the app.

Refunds

Payments and refunds are handled by Apple. To request a refund, visit reportaproblem.apple.com.

Parent Questions

Purchases are protected by a parental math gate before checkout. If you have a concern, email us and we will help.

Response Time

We try to respond to support messages as soon as possible. Please include your device model, iOS version, and a short description of the issue.
